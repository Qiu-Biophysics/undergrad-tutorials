# Code of Conduct

This repository is a learning space. Please keep discussions respectful, constructive, and welcoming.

Expected behavior:

- Be kind and patient with beginners.
- Give feedback that helps the work improve.
- Ask questions when something is unclear.
- Respect different levels of experience.

Unacceptable behavior:

- Harassment or personal attacks
- Dismissive or insulting comments
- Sharing private data without permission
- Publishing another person's work without credit

If there is a concern, contact a repository maintainer or organization owner.
